# Prompt 05 — Non-negotiables

仕様がブレたら、必ずこの文書を再確認してください。

## このゲームは何か

これはGame Boy向けのRogueクローンです。Rogue風のカジュアルダンジョンRPGではありません。

## 簡略化してはいけないもの

- 永久死
- 未識別アイテム
- 空腹
- 呪い
- 罠
- ターン制
- 部屋と通路
- 階層移動
- 限られた情報で判断する緊張感
- 斜め移動
- 持ち物管理
- メッセージログ

## 変えてよいもの

- 画面表示方法
- スクロール方式
- 入力方式
- 内部データ構造
- メモリ圧縮方法
- タイル絵
- メニューUI
- 一部の同時出現数
- 表示名

## 最初はASCIIでよい

最初からドット絵を描かない。ASCII風タイルで作り、ゲームとして成立したあと、8x8ドット絵へ差し替える。

## Claude Codeへの注意

勝手に以下をしないこと。

- 現代的なローグライトにしない
- スキルツリーを追加しない
- 町や店を追加しない
- ストーリー重視にしない
- オートマップを親切にしすぎない
- 難易度を下げるために識別や呪いを消さない
- アイテム種類を大幅削減しない
