# Controls(実装済み仕様)

風来のシレンGB風。prompts/06_controls_spec.md の案Cベース(斜め=同時押し)。

## ワールド画面

| 入力 | 動作 |
|---|---|
| 十字キー | 4方向移動(1歩=4フレームのグライド表示)/ 敵に向かえば攻撃(bump) |
| 十字 同時押し | **斜め移動**(最初の入力から3フレームの猶予窓で合成)。角抜け不可・ドアの斜め出入り不可(Rogue準拠) |
| 十字 長押し | オートリピート移動(8F遅延→設定間隔)。**リピート中はグライドが倍速** |
| A | 階段の上なら昇降 / それ以外は足元・周囲をサーチ(罠発見、1ターン) |
| B + 十字 | **ダッシュ**(グライドなしのスナップ移動、約5フレーム/歩)。敵可視・アイテム・ドア・階段・発見済み罠・通路分岐・部屋の出入り・被弾で停止。オートセーブはダッシュ終端で1回 |
| A + B(同時押し) | 足踏み(1ターン待つ)。**押し続けると0.5秒後からリピート**(足踏み連続、アニメ倍速) |
| SELECT(タップ) | 持ち物 |
| SELECT(約0.4秒長押し) | **全体マップ**(1マス=2×2ドットの俯瞰。探索済みのみ表示。開いたらSELECTを離しても保持し、**Bで閉じる**) |
| START 押しっぱなし + 十字2方向 | **斜め固定移動**(縦横は無視、斜めのみ。誤入力で縦横に暴発しない) |
| START(単押し) | メインメニュー(押している間に十字を触らず離したときに開く) |

## 持ち物画面(SELECT)

- 十字上下: カーソル / A: 使う・読む・食べる・装備/解除 / SELECT: 置く / B: 閉じる
- 杖・投擲は方向入力プロンプト(十字で8方向、B キャンセル)

## メニュー(START)

Rest a turn / Search / Throw-fire / Message log / Display mode(ASCII⇔GFX)/
Repeat speed(1=遅 2=普通 3=速)/ **Language(英語⇔日本語)**/ Save & quit

- 日本語モード: メッセージ・メニュー・アイテム名・モンスター名がかな表記
  (美咲フォント 8×8)。ステータス行の略号(HP/St/AC等)は共通。
  設定はセーブに保存される。

- **1歩ごとに自動セーブ**。電源断・中断はいつでも安全、そして
  「やり直し」は存在しない(全ての一歩が確定)。死亡・勝利でセーブは破棄。
- Save & quit はタイトルへ戻るだけ(セーブは常に最新)。SELECT で再開。

## 戦闘フィードバック

- ダメージを**受けた**: 自分のマスの背景が**赤**に2回点滅。同時に
  ステータス行の **HP表示も赤**に点滅
- ダメージを**与えた**(近接・投擲・杖): 相手のマスの背景が**黄**に2回点滅
- GBC はBG属性パレット(4=赤 / 5=黄)で表現。DMG では対象スプライトが
  反転パレット(OBP1)で明滅する

## メッセージ行

新しいメッセージは**下から迫り出し、古い行が上に押し出される**スライド
(2px×4フレーム)。スライドは**行動アニメーションとフラッシュの完了後**に
開始され、**ノンブロッキング**(スライド中も入力・行動可能。次の行動や
メッセージが来たら即スナップして続行)。メッセージ行は専用の固定20タイル
(VRAM 63..82)にピクセルストリーミングされる。

## 誤操作防止

- 中断はメニュー最下段(カーソル移動が必要)
- 呪われた装備の解除試行はターンを消費して発覚メッセージ
- ポップアップ(重要イベント)は A/B で明示的に閉じる
